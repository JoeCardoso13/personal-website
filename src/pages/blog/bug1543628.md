## A 3-Layers Solution

### 1. Modifying the component

Rename the AccessibleImage component to just Image or DebuggerImage (it doesn't have any accessibility feature; we could add the ability to pass alt text, though.)

### 2. Changing the base class

Change the base class from img to something a bit less generic, for instance dbg-img.

### 3. Changing class names

Change all the image specific class names from <name> to dbg-img-<name>, for instance: dbg-img-worker.
